# E-Commerce Website (MERN Stack)

This is a simple E-Commerce project built using the MERN stack.

## How to Run

### Backend
1. cd backend
2. npm install
3. Create .env file from .env.example
4. node server.js

### Frontend
1. cd frontend
2. npm install
3. npm start

Frontend: http://localhost:3000
Backend: http://localhost:5000
